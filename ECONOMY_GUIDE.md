# LegacyLand 经济系统使用指南

## 系统概述

基于 eco.txt 设计的虚拟国家经济系统，实现了实体货币本位制度。

### 核心特性

- **实体货币**：纸张 + 自定义 NBT 标签
- **国库印章**：特殊物品，用于铸造官方货币
- **基准货币（SBC）**：金锭作为储备金
- **汇率系统**：基于储备金和信用系数动态计算
- **独立数据库**：economy.db（SQLite）

## 快速开始

### 1. 创建国库

**前置条件**：
- 必须是 Towny 国家的国王
- 国家还没有国库

**命令**：
```
/economy create
```

国库将在你当前位置创建，记录坐标和世界信息。

### 2. 获取国库印章

**命令**：
```
/economy seal
```

你将获得一个特殊的"国库印章"物品，这是铸造货币的必需品。

**特性**：
- 无法在生存模式获取（只能通过命令）
- 绑定到特定国家
- 用于防伪认证

### 3. 铸造货币

**材料需求**：
- 1x 纸张（Paper）
- 1x 墨囊（Ink Sac）
- 持有国库印章

**命令**：
```
/economy mint <面值>
```

**示例**：
```
/economy mint 100
```

这将消耗材料并生成一张面值 100 金锭的货币。

### 4. 存入储备金

**命令**：
```
/economy deposit <数量>
```

将手持的金锭存入国库作为储备金（SBC）。储备金越多，货币汇率越高。

**示例**：
```
/economy deposit 64
```

### 5. 提取储备金

**命令**：
```
/economy withdraw <数量>
```

从国库提取金锭。需要国王权限。

### 6. 查看国库信息

**命令**：
```
/economy info [国家名]
```

显示：
- 国库位置
- 储备金（SBC）数量
- 已发行货币总量
- 当前汇率
- 信用系数

### 7. 查看汇率

**命令**：
```
/economy rate [国家名]
```

显示指定国家货币对 SBC（金锭）的汇率。

## 货币物品

### 货币属性

每张货币包含以下 NBT 数据：
- **国家名称**：发行国
- **面值**：等值多少金锭
- **序列号**：唯一标识符（格式：国家-时间戳-随机数）
- **发行时间**：铸造时间戳

### 显示信息

```
§6§l中华帝国 货币
§7面值: §e100.00 §7金锭
§7序列号: §fCHINA-1234567890-ABC123
§8官方发行 · 防伪认证
```

### 防伪机制

- 所有货币记录在数据库中
- 序列号唯一且可追溯
- 无法通过普通方式复制

## 汇率计算

### 公式

```
汇率 = (国库储备金 / 已发行货币总量) × 信用系数
```

### 示例

假设：
- 国库储备金：1000 金锭
- 已发行货币：5000 面值
- 信用系数：1.0

```
汇率 = (1000 / 5000) × 1.0 = 0.2
```

即：1 单位货币 = 0.2 金锭

### 信用系数（α）

影响因素（未来实现）：
- 国家在线时长
- 领地面积
- 历史违约记录
- 战争胜负

## 经济策略

### 稳定货币

**增加储备金**：
- 多挖金矿
- 通过贸易获取金锭
- 存入国库

**控制发行量**：
- 不要过度铸币
- 回收旧币

### 货币贬值风险

**原因**：
- 储备金不足
- 过度发行货币
- 信用系数下降

**后果**：
- 汇率降低
- 购买力下降
- 国际贸易受损

### 货币升值策略

**增加储备**：
- 大量存入金锭
- 提高储备金比例

**减少流通**：
- 回收市场上的货币
- 销毁旧币

## 常见问题

### Q: 货币会消失吗？

A: 不会。货币是实体物品，只要不被销毁就会一直存在。

### Q: 可以伪造货币吗？

A: 不能。所有货币都有唯一的 NBT 标签和数据库记录，无法复制。

### Q: 汇率会变化吗？

A: 会。每次存入/提取储备金或铸造货币时，汇率都会重新计算。

### Q: 国库被破坏怎么办？

A: 国库只是一个坐标记录，不是实体方块。即使位置被破坏，数据仍然保存在数据库中。

### Q: 可以创建多个国库吗？

A: 不能。每个国家只能有一个国库。

## 技术细节

### NBT 标签

```java
currency: "true"           // 标识为货币
nation: "中华帝国"          // 国家名称
denomination: 100.0        // 面值
serial: "CHINA-123-ABC"    // 序列号
issued_at: 1234567890      // 发行时间戳
```

### 国库印章 NBT

```java
currency: "seal"           // 标识为印章
nation: "中华帝国"          // 国家名称
serial: "SEAL-UUID"        // 唯一标识
```
