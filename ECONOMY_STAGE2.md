# 经济系统阶段 2：银行与跨国交易

## 新增功能

### 1. 银行系统（M2 货币）

**电子存款**：将实体货币存入银行，转换为数字余额。

**优势**：
- 不占用背包空间
- 可以快速转账
- 支持跨国兑换

### 2. 跨国交易

**汇率转换**：不同国家货币可以相互兑换。

**关税系统**：跨国兑换收取 15% 手续费。

## 命令使用

### /bank deposit
**功能**：存款 - 将手持的货币存入银行

**用法**：
1. 手持货币物品
2. 执行 `/bank deposit`
3. 货币消失，余额增加

**示例**：
```
手持 100 面值的中华帝国货币
/bank deposit
→ 成功存入 100.00 中华帝国 货币！
→ 当前余额: 100.00
```

### /bank withdraw <金额> [国家]
**功能**：取款 - 将电子余额转换为实体货币

**用法**：
```
/bank withdraw 50
/bank withdraw 100 中华帝国
```

**示例**：
```
/bank withdraw 50
→ 成功取出 50.00 中华帝国 货币！
→ 剩余余额: 50.00
→ 获得实体货币物品
```

### /bank balance [国家]
**功能**：查询余额

**用法**：
```
/bank balance              # 查询所有国家余额
/bank balance 中华帝国      # 查询指定国家余额
```

**示例**：
```
/bank balance
========== 银行余额 ==========
中华帝国: 1000.00
日本帝国: 500.00
============================
```

### /bank transfer <玩家> <金额> [国家]
**功能**：转账给其他玩家

**用法**：
```
/bank transfer Steve 100
/bank transfer Alex 50 中华帝国
```

**示例**：
```
/bank transfer Steve 100
→ 成功转账 100.00 给 Steve！
→ 剩余余额: 900.00
```

### /bank exchange <源国家> <目标国家> <金额>
**功能**：跨国货币兑换

**用法**：
```
/bank exchange 中华帝国 日本帝国 100
```

**计算公式**：
1. 源货币转 SBC：`100 × 中华帝国汇率`
2. 扣除关税（15%）：`SBC × 0.85`
3. 转目标货币：`扣税后SBC ÷ 日本帝国汇率`

**示例**：
```
假设：
- 中华帝国汇率：1 NC = 0.5 SBC
- 日本帝国汇率：1 NC = 0.4 SBC

/bank exchange 中华帝国 日本帝国 100
→ 100 NC × 0.5 = 50 SBC
→ 50 SBC × 0.85 = 42.5 SBC（扣除15%关税）
→ 42.5 SBC ÷ 0.4 = 106.25 日本帝国货币

成功兑换！
扣除: 100.00 中华帝国
获得: 106.25 日本帝国
关税: 7.50 SBC (15%)
```

## 工作台合成货币

### 配方
```
[国库印章] + [纸张] + [墨囊] = 货币
```

### 使用流程
1. 在工作台放入：国库印章 + 纸张 + 墨囊
2. 系统提示输入面值
3. 在聊天框输入数字（如：100）
4. 自动生成货币物品

### 注意事项
- 必须是该国家的成员
- 国库印章不会被消耗
- 纸张和墨囊会被消耗
- 输入 `cancel` 取消铸造

## 经济策略

### 套利机会

如果两国汇率差异大，可以通过兑换套利：

**示例**：
- A 国汇率：1 NC = 0.8 SBC
- B 国汇率：1 NC = 0.5 SBC

**操作**：
1. 用 100 A 国货币兑换 B 国
2. 100 × 0.8 = 80 SBC
3. 80 × 0.85 = 68 SBC（扣税）
4. 68 ÷ 0.5 = 136 B 国货币

**结果**：获得 36% 收益（扣除关税后）

### 关税影响

**进口成本**：
- 跨国购买商品需支付 15% 额外费用
- 鼓励使用本国货币交易

**贸易保护**：
- 保护本国经济
- 增加国库收入（关税归国库）

### 汇率操纵

**提高汇率**：
1. 增加储备金（存入更多金锭）
2. 减少货币发行量

**降低汇率**：
1. 大量铸造货币
2. 减少储备金
